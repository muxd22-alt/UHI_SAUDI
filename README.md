# 🇸🇦 إطار الاقتصاد المؤتمت السعودي

*(Saudi Automated Economic Framework · Key Inputs from the World Bank)*

هذا المشروع عبارة عن محاكي تفاعلي متطور، مصمم خصيصاً لنمذجة التحول الاقتصادي الكلي للمملكة العربية السعودية في ظل تحولات سوق العمل الجذرية الناجمة عن ثورة الذكاء الاصطناعي (AI) والأتمتة المتقدمة.

يستخدم المحاكي محركاً رياضياً حتمياً لتقدير ورسم سيناريوهات المستقبل؛ مستهدفاً تلك اللحظة التي يحل فيها تراكم الثروة السيادية، وبرامج دعم الدخل الأساسي، محل الاقتصاد التقليدي القائم حصراً على الأجر الوظيفي.

---

## ⚙️ البنية التقنية الأساسية

- **التغذية الحية بالبيانات**: يعتمد المحاكي على جلب مسارات البيانات الحية من البنك الدولي (World Bank API) كخط أساس لعدد السكان والناتج المحلي.
- **واجهة رشيقة ومستقلة**: تطبيق الواجهة الأمامية (`docs/index.html`) يعمل بالكامل وبشكل مستقل على المتصفح ولا يتطلب أي خوادم خلفية (Zero-Backend).
- **التكيف اللغوي والعملات (Bi-Directional)**: تقنية تبديل حي بين الواجهة الإنجليزية (والتي تعمل بالدولار الأمريكي كمقياس عالمي) وبين الواجهة العربية الفاخرة التي تعيد ضبط المحرك الحسابي ليعرض كافة المخرجات بالريال السعودي (بمعامل `x3.75`).

---

## 🎛️ دليل المتغيرات والمدخلات (Sliders)

قائمة شاملة تشرح كل عنصر تحكم داخل لوحة القيادة في المحاكي:

### 1️⃣ نطاق المحاكاة والنمو العام
| المتغير | الوصف التقني |
|---------|-------|
| 📅 **سنة النهاية** | تاريخ انتهاء المحاكاة الجارية ورسم النطاق الزمني للمنحنيات. |
| 🤖 **وتيرة الأتمتة** | النسبة المئوية لمهام العمل البشرية التي تُستبدل بالذكاء الاصطناعي في كل عام (انكماش الوظائف). |
| 📈 **نمو الناتج المحلي** | معدل نمو الناتج المحلي الإجمالي السنوي الأساسي للدولة، بمعزل عن طفرات الأتمتة. |

### 2️⃣ صندوق الاستثمارات العامة (PIF) 
| المتغير | الوصف التقني |
|---------|-------|
| 💰 **الأصول المدارة** | رأس المال الأساسي وصافي الأصول المدارة لصندوق الاستثمارات العامة (أرقام أولية). |
| 🇸🇦 **المساهمة من الناتج** | النسبة المئوية من إجمالي الناتج المحلي التي تقتطعها الدولة وتضخها كاستثمار في الصندوق سنوياً. |
| 📊 **معدل العائد** | متوسط العائد السنوي المتوقع (ROI) للمحفظة الاستثمارية العالمية والمحلية للصندوق. |
| 🏛️ **قاعدة الإنفاق المدني** | النسبة المستقطعة سنوياً من إجمالي أصول الصندوق وتوزيعها كأرباح (Dividends) مباشرة للمواطنين. |

### 3️⃣ حساب المواطن (الدخل الأساسي)
| المتغير | الوصف التقني |
|---------|-------|
| 💵 **الهدف الشهري** | المعادل بالريال للإعانة الشهرية الأساسية الممنوحة كدخل ثابت ومضمون لكل فرد. |
| ⏳ **سنوات التدرج** | الفترة الزمنية بالسنوات (المدة الممنوحة) لرفع قيمة الدعم الحالي تدريجياً للوصول للهدف. |

### 4️⃣ تداول وعوائد البيانات (الامتيازات)
| المتغير | الوصف التقني |
|---------|-------|
| 💹 **نمو الأسهم المحلية** | معدل العائد الافتراضي لتوسع ملكية المواطنين بالأسهم الخاصة للشركات عبر السوق المالية (تداول). |
| 🌐 **أقصى عائد للبيانات** | الحد الأقصى للعوائد المدفوعة للأسرة مقابل استخدام بياناتها الشخصية والوطنية في تدريب نماذج الذكاء الاصطناعي الذاتية للدولة. |

### 5️⃣ منح الشباب وعوائد الطاقة
| المتغير | الوصف التقني |
|---------|-------|
| 💼 **مبلغ التأسيس للشباب** | منحة رأس مال تأسيسية تقدم للمواطن تلقائياً عند الولادة وتتراكم أرباحها لصرفها عند بلوغ سن 18 عاماً (نطاقات استثمارية). |
| 🛢️ **توزيعات الطاقة** | شريحة أرباح سنوية مباشرة تُوزع نقداً للأسر من بيع موارد النفط والطاقة والموارد المعدنية. |

### 6️⃣ التأثيرات الانكماشية للذكاء الاصطناعي (Deflationary Effects)
| المتغير | الوصف التقني |
|---------|-------|
| 📉 **أقصى انخفاض للأسعار** | محاكاة للانهيار الإيجابي لتكاليف قطاعات النقل، الطب، القانون، والطاقة استجابة لانفجار إنتاجية الذكاء الاصطناعي المعمم (AGI). هذا الانكماش يضاعف القوة الشرائية الحقيقية للدخل الممنوح للمواطن دون زيادة التضخم. |

## 🇬🇧 English System Summary
This framework acts as a highly-tuned macroeconomic dashboard designed explicitly for Saudi Arabia. It maps the timeline transition from a labor-intensive wage economy towards an asset-based sovereign framework utilizing actual baseline vectors from the World Bank API.

**Features Native Currency Scaling:** The deep JS engine computes all heuristics internally using USD ($). Upon switching to the Arabic UI, an intelligent rendering overlay intercepts every localized value, flawlessly scaling datasets and UI sliders to standard **SAR** (Saudi Riyals) instantly with exact decimal-clean bounds.

**Deployment:** Fully stateless static files. Serve `docs/index.html` locally using any standard HTTP server (`python -m http.server 8000`) or deploy natively to GitHub Pages pointing securely to the root `/docs/` directory.
